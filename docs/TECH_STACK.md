# Technology Stack

HealthRisk AI utilizes a bleeding-edge, production-ready technology stack combining the best of the JavaScript and Python ecosystems.

## Frontend (Client & UI)

| Technology | Purpose |
|------------|---------|
| **Next.js 16** | Core React framework leveraging the App Router and Turbopack for fast builds. |
| **React 19** | UI library for building component-driven interfaces. |
| **TypeScript** | Static typing to ensure robust, error-free code during development. |
| **Tailwind CSS 4** | Utility-first CSS framework for rapid, responsive styling. |
| **Framer Motion** | Physics-based animation library for micro-interactions and page transitions. |
| **Recharts** | Composable charting library built on React components for the Analytics dashboard. |
| **Lucide React** | Beautiful, consistent icon set. |

## Backend (API & Machine Learning)

| Technology | Purpose |
|------------|---------|
| **FastAPI** | High-performance asynchronous Python web framework for building APIs. |
| **Python 3.13** | Core backend language. |
| **Pydantic** | Data validation and settings management using Python type annotations. |
| **Scikit-learn** | Machine learning library used to train and serialize the predictive models. |
| **Pandas / NumPy** | Data manipulation and numerical operations for preprocessing inputs. |
| **Joblib** | Lightweight pipelining to load pre-trained ML models efficiently. |
| **Uvicorn** | Lightning-fast ASGI server implementation to run the FastAPI app. |

## Development & Deployment Tools

| Technology | Purpose |
|------------|---------|
| **Git / GitHub** | Version control and source code management. |
| **ESLint / Prettier** | Code linting and formatting standards. |
| **Vercel** | (Target) Hosting for the Next.js frontend with Edge network capabilities. |
| **Render / Railway** | (Target) Hosting for the FastAPI backend. |
| **Turbopack** | Next.js incremental bundler for ultra-fast local development. |
